NLP - Assignment 1 (Data Extraction)

This assignment uses Selenium to scrape case judgments from the Supreme Court of Pakistan website.

Features:
- Extracts case details (subject, case no, title, judge, dates).
- Downloads available judgment PDFs into /judgementpdfs.
- Saves structured JSON file (SupremeCourt_G4.json) with metadata.
- Includes CLI outputs for tracking progress.

Requirements:
- Python 3
- Selenium
- Requests
- Chrome WebDriver (auto-installed using webdriver-manager)
